Assetinventory

Tool which runs on web2py python webui which allows for easy inventorying utilizing nessus and NMAP.

Download as .zip Download as .tar.gz View on GitHub

Welcome to Shadey Security's Simple Asset Inventory System

This system of scripts is designed to make it super easy for your personal or enterprise inventorying needs! Continue to checkback and provide issues with ideas, we listen as long as we have time!

Background

The need for this tool was initially internally, and very brittle. We decided to migrate it to open source predominantly for two reasons:

  • Github is free, we like free more than we do open-source, which we love
  • It allows people outside of the small scripting duo it was originally in to work on it
